# Noraebang
Noraebang is the first Korean style karaoke booth in Finland, where you can sing alone or with a few other friends in a private space, without any pressure of an audience or need to drink alcohol.

It is placed on the first floor of Abloc building, by K-market (Adderss Otaniementie 12, 02150 Espoo).

Its structure was created using a converted Finnish wooden sauna and soundproofed through bio-base materials like wood-fiber foam and mycelium.

Mycelium soundproofing panels were designed through participatory workshops using 3d-printed Hangul (Korean writing system) inspired tiles, hosted by BioMakers Lab. Over 50 people from Aalto community and beyond participated.
